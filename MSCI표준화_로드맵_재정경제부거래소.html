MSCI 선진국 편입 5대 과제 — 표준화 로드맵

재정경제부·거래소(KRX)·예탁원(KSD)·증권사 ‘데이터 연결표준’ 실행안
기준일 2026-06-30 · 신규입법 없이 표준·고시·업무규정 수준에서 해소 · 금융세제 소득성격 표준과 통합
설계 원칙 — ① 펀더멘털이 아닌 ‘연결표준 결손’ 해소이므로 신규입법 최소화(표준·고시·업무규정). ② 이미 보유한 데이터(거래소·예탁원·증권사)를 연결하는 단일 표준을 세운다. ③ ④배당 표준은 금융세제 소득성격(배당/자본환급) 코드와 ‘하나의 표준’으로 통합해 MSCI 개선과 ROC 이중과세 해소를 동시 달성한다(통섭·통합).

1. 3단계 로드맵 (24개월, 무입법 우선)

단계기간목표산출
P1 정의·합의0–6개월5개 과제별 ‘연결표준’ 정의 + 기관 합의표준안·업무규정 초안·거버넌스 체계
P2 연결·파일럿6–12개월거래소–예탁원–증권사 데이터 연결 시범파일럿 결과·소득성격 코드 시범
P3 정착·재신청12–24개월전면 시행·트랙레코드 축적MSCI 재제출 + ROC 이중과세 해소 동시

핵심: MSCI는 ‘개혁의 정착·트랙레코드’를 요구하므로, P1·P2를 빨리 끝내고 P3에서 ‘실효 데이터’를 쌓는 것이 관건.

2. 5대 사유 → 표준화 과제 매핑

# 사유표준화 과제(무입법)주관 / 협조단계
① 역외 원화외환 연장거래 정착·역외 청산결제 인프라(RFI 등록 안착)·FX 메시지 표준(ISO 20022)재정경제부·한국은행 / 은행P1–P3
② 등록·계좌옴니버스(통합)계좌 전면 정착·현물이전/장외 표준절차·LEI 식별표준금융위·KSD / 증권사P1–P2
③ 거래소 데이터시장데이터 라이선스 표준개방·반경쟁 조항 정비·해외 상품조성 허용KRX / 금융위P1–P2
④ 배당 절차 ★배당 선확정 의무·정착 + ‘분배 소득성격 코드’(배당/CG/ROC) 표준 통합재정경제부·금융위·KSD / 상장사P1–P3
⑤ 공매도불법공매도 중앙감시 운영표준 간소화·차입/잔고 데이터 표준연결금융위·KRXP1–P2
  • 주목 — ④는 별표(★): 유일하게 ‘세제’와 직접 맞물리는 과제. 배당락 前 배당 확정 + 분배 소득성격 코드(배당·자본이득·자본환급)를 한 표준으로 만들면, MSCI ④와 ROC 과세 둘 다 해결.

3. ④ 통합표준 상세 — ‘분배정보 표준’ 하나로 두 문제 동시 해결

MSCI 배당 과제와 금융세제 소득성격 단절은 모두 ‘분배(distribution) 정보의 표준·연결’ 문제다. 단일 표준으로 묶는다.

요소MSCI ④ 충족세제(소득성격) 충족
배당 기준일·금액 선확정글로벌 표준 부합정보 적시성 확보
분배 소득성격 코드(배당/CG/ROC)분배정보 투명·표준ROC=취득원가 환급 정상처리
발행사→QI→KSD→증권사 연결데이터 일관성소득성격 단절 해소(이중과세 방지)
  • 연결 — 발행사 1099-DIV Box3·Form 8937·ICI의 소득성격을 QI(외국보관기관)가 투입 → KSD가 표준코드로 보존 → 증권사가 원천징수·신고에 반영. 현재 KSD/증권사 단계의 ‘단절’만 표준으로 메우면 됨.

4. 거버넌스 & 추진체계

  • 총괄 — 재정경제부(금융세제·외환제도) 주관, 금융위·KRX·KSD·증권사·한국은행 참여 ‘데이터 연결표준 협의체’.
  • 수단 — 법률 개정이 아닌 ‘표준·고시·업무규정·거래소 규정’ 수준에서 우선 처리(속도·반발 최소화).
  • 문서원칙 — 권한 추궁이 아니라 ‘이미 가진 데이터의 정합·연결’이라는 비용재배치 프레임으로 제시(결재권자가 정정·정합으로 수용하기 쉽게).

사유대표 KPI선진국 기준

③ 데이터데이터 라이선스 발급·해외상품 조성 수제약 없는 상품화
