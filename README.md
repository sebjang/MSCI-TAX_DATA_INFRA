# 한국 자본시장 데이터 인프라 분석

**MSCI 선진국 편입 낙제**와 **금융세제 데이터 인프라 미비**(소득성격 단절)를 나란히 분석하고, 둘을 한 번에 푸는 **표준화 로드맵**을 제시합니다.

> **핵심 명제** — 한국 자본시장의 두 ‘낙제’는 펀더멘털 문제가 아니라 *“데이터는 이미 있는데 연결표준이 없다”*는 동일한 병리다. 둘 다 신규입법보다 **표준화·연결**로 풀리며, 공교롭게 양쪽 모두 **‘배당(분배) 데이터’**가 공통의 약한 고리다.

## 🔑 한눈에

### MSCI 선진국(DM) 편입 5대 낙제 사유 (2026 연례검토)
1. **역외 원화시장 부재** — 원화 역외 실물인도 불가·연장시간 유동성 부족
2. **투자자 등록·계좌 경직** — 식별체계 경직, 현물이전·장외거래 제한
3. **거래소 데이터 사용 제한** — 한국지수 기반 투자상품 조성 제약
4. **배당 절차 미정착** — 배당락 前 배당액 확정 시행 상장사 소수 ★
5. **공매도 운영 부담** — 불법공매도 감시체계 운영 부담 과중

### 금융세제 데이터 인프라 미비 — 소득성격 단절
소득성격 데이터는 **3층에 이미 존재**하나 국내 연결단계에서 끊긴다.

```
① 발행사(미국)        1099-DIV Box3 · Form 8937 · ICI   → 원천에 존재
② 외국보관기관(QI)     소득성격을 분배에 투입(재분류)      → 중간에 존재
③ KSD · 국내 증권사    원본 미수집 → 소득성격 단절         → 연결 결손(병목)
```
결과: 자본환급(ROC)이 배당으로 과세되어 **이중과세·취득원가 미반영**.

### 공통 처방
- 이미 보유한 데이터를 연결하는 **단일 표준(분배정보 표준)** — 배당 선확정 + 소득성격 코드(배당/CG/ROC)
- **MSCI ④ 개선**과 **ROC 이중과세 해소**를 *동시* 달성
- 거버넌스: **재정경제부 총괄**, 금융위·KRX·KSD·증권사·한국은행 협의체 — 표준·고시·업무규정 수준(**무입법 우선**)

## 🗺️ 표준화 로드맵 (24개월)

| 단계 | 기간 | 목표 |
|---|---|---|
| **P1 정의·합의** | 0–6개월 | 5개 과제별 연결표준 정의 + 기관 합의 |
| **P2 연결·파일럿** | 6–12개월 | 거래소–예탁원–증권사 데이터 연결 시범 |
| **P3 정착·재신청** | 12–24개월 | 전면 시행·트랙레코드 → MSCI 재제출 + ROC 해소 |
